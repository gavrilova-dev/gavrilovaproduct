# Gavrilova Product Portfolio

Персональный сайт-портфолио Product Designer / Product Manager.

🔗 Website: https://gavrilovaproduct.ru

## О проекте

Сайт создан как продуктовый портфолио-проект, который показывает не только финальные интерфейсы, но и процесс принятия решений: исследование проблемы, UX-подход, продуктовую логику и результат.

Цель проекта — показать опыт работы с цифровыми продуктами, мобильными приложениями и дизайн-системами.

## Кейсы

На сайте представлены продуктовые кейсы:

- **Редизайн главной страницы**
  Редизайн главного экрана мобильного приложения с фокусом на сценарии пользователя и быстрый доступ к ключевым действиям.

- **Переработка сценария пополнения**
  Оптимизация сценария пополнения баланса и сокращение количества действий пользователя.

- **Поддержка нескольких номеров**
  Проработка сценария управления несколькими номерами в одном аккаунте.

- **UI Kit и основы дизайн-системы**
  Создание UI Kit и дизайн-системы: компоненты, токены, переменные, Auto Layout.

- **Обновлённое банковское приложение**
  Пусть от создания нового банковского приложения на Flutter до вывода в продакшн.

## Технологии

Проект развернут самостоятельно:
